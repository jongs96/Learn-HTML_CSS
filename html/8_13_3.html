CSS 속성 선택자 7종 실습

각 선택자가 적용되면 배경/테두리 색으로 강조됩니다.

1) [속성] — 해당 속성이 "존재"하면 선택



2) [속성="값"] — "정확히 일치"

3) [속성~="값"] — 공백으로 구분된 "단어" 중 포함

class="row beta" → 적용 O
class="row alpha" → 적용 X
class="row alpha beta gamma" → 적용 O

4) [속성|="값"] — 값 "그 자체" 또는 "값-"으로 시작

lang="en" → 적용 O
lang="en-US" → 적용 O
lang="fr" → 적용 X

5) [속성^="값"] — 값으로 "시작"

https로 시작 → 적용 O
http로 시작 → 적용 X
mailto: → 적용 X

6) [속성$="값"] — 값으로 "끝남"

data-file="photo.png" → .png로 끝나므로 적용 O
data-file="report.pdf" → 적용 X
data-file="sprite.svg" → 적용 X

7) [속성*="값"] — 값에 "포함" (위치 무관)

href에 "naver" 포함 → 적용 O
쿼리 문자열에 "naver" 포함 → 적용 O
"naver" 미포함 → 적용 X
