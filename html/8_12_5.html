<input> type 속성 실습

1) 텍스트 입력 — placeholder / required

※ placeholder는 힌트일 뿐 값이 아니고, required는 “값 비우면 제출 불가”를 의미합니다.

2) 이메일 — 형식 검사 + required

※ 잘못된 형식이면 브라우저가 기본 경고를 띄웁니다.

3) 날짜 — min / max 범위 제한

※ 범위를 벗어난 날짜는 브라우저가 막습니다.

4) 체크박스 — 최소 1개 필수 (최대 개수 제한은 HTML만으로 불가)

※ 최소 1개 이상은 required로 가능. 최대 개수 제한은 자바스크립트 필요.

5) Range 슬라이더 — 기본값 / 범위 / 간격

※ value로 기본값을 정하고, min/max/step으로 범위와 간격을 제어합니다.

(GET 방식이라 제출 시 주소창에 값이 표시되어 비교하기 쉬워요)

유용성 체크 (HTML만 기준)

  • ✅ placeholder로 입력 힌트 표시 가능 (값은 아님)
  • ✅ required로 “필수 입력” 강제 가능
  • ✅ 날짜 min/max로 선택 가능한 범위 제한 가능
  • ✅ range의 기본값(value), 범위(min/max), 간격(step) 설정 가능
  • ✅ 체크박스 그룹의 “최소 1개 이상 선택”은 required로 가능
  • ❌ 체크박스 그룹의 “최대 N개 선택”은 HTML만으로 불가 (자바스크립트 필요)
